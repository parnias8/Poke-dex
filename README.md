# PokedeX

A modern Pokédex web application built with **Next.js** and **Tailwind CSS**.

## Features

- Browse Pokémon
- Search Pokémon by name
- View detailed Pokémon pages
- See types, abilities, strengths, weaknesses, and evolution
- Responsive design

## Tech Stack

- Next.js
- React
- Tailwind CSS
- JavaScript

## Run Locally

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```
